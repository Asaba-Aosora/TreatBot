临床试验匹配 Demo 输入页面

请填写匹配所需属性，系统会自动校验格式，并生成可复制的结构化患者输入。

可从建议列表选择，也可输入关键词。若不确定，请选择最接近的病种。
请输入数字，系统会校验范围。
如果不清楚，可先选择“不限”。
请输入整数，表示已接受的治疗线数。
请选择省份并输入城市，再点击“确认省市”。
多个标志物请用逗号分隔。

提示：

  • 地点采用省+市组合可减少输入错误。
  • 疾病标签提供了已有试验中的常见写法，若不完全匹配可输入近似名称。
  • 生成后的 JSON 可直接复制到 `run_match.py` 中的 `build_patient_input(...)` 参数。
